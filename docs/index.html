TechNewsDaily Bot

TechNewsDaily is an automated bot that generates short-form tech news videos from publicly available articles.

The bot:

  • Scrapes tech news from various RSS feeds and news sources
  • Uses AI to generate engaging scripts
  • Creates vertical videos (1080x1920) with captions
  • Automatically posts to TikTok and YouTube

This is a personal automation project for educational and informational purposes.
